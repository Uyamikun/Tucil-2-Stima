Tugas Kecil 2 Strategi Algoritma

#Tentang
Program ini adalah contoh Topological Sort yang menggunakan Algoritma Decrease and Conquer.
Cara kerja Topological Sort pada kode ini yaitu :
1. Mencari simpul dengan derajat masuk nol
2. Hapus simpul tersebut
3. Hapus sisi keluar dari simpul tersebut ke simpul lain/sisi masuk dari simpul tersebut pada simpul lain/sisi
4. Masukkan simpul berderajat nol tersebut pada sebuah array
5. tampilkan simpul tersebut ke layar
6. ulangi langkah 1 sampai 5 sebanyak 8 kali

#Requirement
 Python3

#How to use
Windows:
	1. Extract file zip
	2. Pergi ke folder test
	3. Copy file pada folder test ke dalam folder bin
	4. Jalankan main.exe
Linux:
	1. Extract file zip
	2. Pergi ke folder test
	3. Copy file pada folder test ke dalam folder src
	3. Buka terminal
	4. Pergi ke direktori folder src dari folder yang telah di extract
	5. Ketikkan "python3 main.py" tanpa tanda kutip
